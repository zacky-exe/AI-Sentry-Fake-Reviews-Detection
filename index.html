How to Use

1

Highlight the comment and press Alt+S to scan the selected comment

Alternatively, you can:

1

Highlight the comment and Right-click on the comment you want to scan

2

Select Scan Selected Comment from the context menu

3

Wait for the analysis result to appear in the popup

Install Now

How to Install

1

Press the install button and wait for the file to be downloaded

2

Go to file browser and find the file you downloaded. Right-Click on it and choose Extract All The Files.

3

Click extension icon and select manage extensions. Turn on the "Developer Mode".

4

Click Load unpacked and Choose the 'AI.SENTRY' unzipped file.

5

Make sure the AI Sentry Fake Reviews Detection is in extension section and is turned on. Click the extension logo to see if the "AI Sentry Fake Reviews Detection" is in your extensions.

6

Open Command Prompt and type 'cd Path unzipped AI.SENTRY then press Enter. Then type 'Python server.py' to activate AI.
